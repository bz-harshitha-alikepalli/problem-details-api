Welcome to Free API Support

Problem Slug Data

Get data for a specific problem slug. This API endpoint provides information about a particular problem based on its slug.

  • http://problem-details-api.onrender.com/data?problemSlug=bz-max-min-stack

Company Data

Get data for a specific company. This API endpoint retrieves information related to a particular company.

  • http://problem-details-api.onrender.com/company-data?companyName=Apple

Student Data

Get data for a specific student. This API endpoint fetches information about a particular student.

  • http://problem-details-api.onrender.com/student-data?studentName=Vikas

Data Included

The API endpoints provide access to the following data:

  • Problem Slug Data: Information about specific problems
  • Company Data: Details related to specific companies
  • Student Data: Information about specific students

Generated Content

The generated content includes:

  • Problem Slug Data: Details about the problem, including outgoing and incoming relationships
  • Company Data: Information about the company, including outgoing and incoming relationships with students
  • Student Data: Details about the student, including outgoing and incoming relationships with companies

Stack Used

  • Backend: Node.js with Express.js
  • Database: Neo4j (Graph Database)
  • Frontend: HTML, CSS (minimal)
